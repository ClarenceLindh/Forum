## Intro
This is a MVP for a social forum web application.
Made with Java, React and SQL.

### Installation
1. Import the SQL file to for example HeidiSQL
2. Start backend with IntelliJ
3. Open frontend-folder and run "npm install"
4. Run "npm start"
